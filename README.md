## Telpo POS system
This project for telpo POS mobile.
###### Using package
```yaml
telpo:
  git: https://github.com/choubsovann/telpo.git
```
######
```dart
import 'package:telpo/telpo.dart';
```
###### Telpo functions
```Dart
Telpo.connect();
Telpo.printImage(Uint8list image);
Telpo.disconnect();
```
